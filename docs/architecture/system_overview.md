# AERIS System Overview

Version: 0.1

## Node Types

1. Emergency Node
2. Relay Node
3. Rescue Node

## Communication Method

LoRa Mesh Networking

## Objective

To provide autonomous emergency communication without dependency on:

- Internet
- Cellular Networks
- WiFi

## MVP Architecture

Emergency Node
     ↓
Relay Node 1
     ↓
Relay Node 2
     ↓
Rescue Node
